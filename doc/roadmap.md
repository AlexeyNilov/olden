# Project roadmap

## Milestone 1: Battlefield model

* Battlefield topology
* Coordinate validation
* Neighbor lookup
* Whole-hex obstacles
* Side-based deployment zones
* Occupancy rules

## Milestone 2: Unit model

* Unit identity
* Unit side
* Unit stack data
* Unit speed
* Multi-hex unit footprint rules

## Milestone 3: Range and movement math

* Distance between two hexes
* Movement radius by unit speed
* Spell area-of-effect rings
* Line-of-sight rules

## Milestone 4: Movement simulation

* Movement validation
* Movement cost rules
* Pathfinding

## Milestone 5: Combat actions

* Attack targeting
* Damage resolution
* Turn-order integration

## Milestone 6: Battlefield view

* Field rendering
* Unit placement display
* Obstacle and deployment-zone display
